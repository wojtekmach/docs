    Specs

    difference(t(val1), t(val2)) :: t(val1) when val1: value(), val2: value()

    Returns a set that is map_set1 without the members of map_set2.

    Examples 

    iex> MapSet.difference(MapSet.new([1, 2]), MapSet.new([2, 3, 4]))

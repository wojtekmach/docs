    Requests.decode

    You're seeing just the function decode, go back to Requests module for more information.
    Link to this function

    decode(response, opts \\ [])

    View Source

    Decodes the response body based on the content-type header.

    Options

    • :json_decoder - if set, used for JSON. Defaults to &Jason.decode/1

    • :csv_decoder - if set, used for CSV. Defaults to &NimbleCSV.RFC4180.parse_string(&1, skip_headers: false)

    • :gzip_decoder - used for gzip. Defaults to &:zlib.gunzip/1

    • :zip_decoder - used for ZIP. Defaults to using :zip.unzip/2

    Examples 

    iex> Requests.get!("https://httpbin.org/json").body
%{...}

iex> Requests.get!("https://httpbin.org/json", json_decoder: fn _ -> "fake" end).body
"fake"

    Requests.encode_request_body

    You're seeing just the function encode_request_body, go back to Requests module for more information.
    Link to this function

    encode_request_body(request, opts)

    View Source

    Encodes the reqeuest body based on the content-type header.

    Options

    • :json_encoder - if set, used on the "application/json*" content type. Defaults to &Jason.encode_to_iodata!/1 if jason dependency is installed.

    • :csv_encoder - if set, used on the "text/csv*" content type. Defaults to &NimbleCSV.RFC4180.dump_to_iodata/1 if nimble_csv dependency is installed.

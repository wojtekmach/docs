    Req.Adapter behaviour (req v0.6.0-dev)

    View Source

    Behaviour for Req adapters — the modules that perform the actual HTTP request.

    An adapter receives the connection and the runtime options keyword and returns either the connection with :status, :resp_headers, :resp_body populated, or {:error, conn, exception} on failure.

    Built-in adapters: Req.FinchAdapter, Req.PlugAdapter.

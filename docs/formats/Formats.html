    decode_xml(data)

    Decodes XML data.

    See also encode_xml_to_iodata/1 for encoding.

    Examples 

    iex> data = ~s{<?xml version="1.0" encoding="utf-8"?><points><point x="1" y="2"/><point x="3" y="4"/></points>}
iex> Formats.decode!(data, :xml)
{"points",
 [
   {"point", %{"x" => "1", "y" => "2"}, []},
   {"point", %{"x" => "3", "y" => "4"}, []}
 ]}

    format(opts)

    Detects a format based on MIME type or filename.

    Options

    • :mime

    • :filename

    Examples 

    iex> Formats.format(mime: "application/x-gzip")
:gzip

iex> Formats.format(mime: "application/vnd.acme+json")
:json

iex> Formats.format(filename: "test.json")
:json

iex> Formats.format(filename: "bad")
nil

    create ( Metadata, Files, Config )

    View Source

    Creates a package tarball.

    Returns the binary of the tarball the "inner checksum" and "outer checksum". The inner checksum is deprecated in favor of the outer checksum.

    Examples:

      > Metadata = #{<<"name">> => <<"foo">>, <<"version">> => <<"1.0.0">>},
  > Files = [{"src/foo.erl", <<"-module(foo).">>}],
  > hex_tarball:create(Metadata, Files).
  {ok, #{tarball => <<86,69,...>>,
         outer_checksum => <<40,32,...>>,
         inner_checksum => <<178,12,...>>}}

    unpack ( Tarball, Output, Config )

    View Source

    Unpacks a package tarball.

    Remember to verify the outer tarball checksum against the registry checksum returned from hex_repo:get_package(Config, Package)`. Examples: ``` > hex_tarball:unpack(Tarball, memory). {ok,#{outer_checksum => <<...>>, contents => [{"src/foo.erl",<<"-module(foo).">>}], metadata => #{<<"name">> => <<"foo">>, ...}}} > hex_tarball:unpack(Tarball, "path/to/unpack"). {ok,#{outer_checksum => <<...>>, metadata => #{<<"name">> => <<"foo">>, ...}}}''

ETF (External Term Format) encoding/decoding support.

Supported types are:

Rust TypeElixir Type
i32integer
f64float
boolboolean
Stringbinary
Vec<T>list(a)
HashMap<K, V>%{k => v}
(T0, T1, ...){a, b, ...}
Option<T>a | nil
Result<T, E>{:ok, a} | {:error, b}
Termterm

Notably, functions, PIDs, ports and references are NOT supported.

Custom types can be supported by deriving ToTerm and FromTerm, for example:

use elixirkit::term::{ToTerm, FromTerm};

#[derive(ToTerm, FromTerm)]
enum Message {
    Ready,
    Echo(String),
}
